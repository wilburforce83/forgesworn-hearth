# Forgesworn Hearth Frontend

This is a simple Vite + TypeScript + vanilla SPA that talks to the existing backend running on `http://localhost:3000`.

## Getting started

1. Install dependencies: `npm install`
2. Run the dev server (default `http://localhost:5173`): `npm run dev`
3. Make sure the backend API is running at `http://localhost:3000` before using the app.
